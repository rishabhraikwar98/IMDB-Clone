## Run Locally

Clone the project

```bash
  git clone https://github.com/rishabhraikwar98/IMDB-Clone
```

Go to the project directory

```bash
  cd IMDB-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
